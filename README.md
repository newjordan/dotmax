# dotmax

High-performance terminal braille rendering for images, animations, and graphics

## Installation

Coming soon to [crates.io](https://crates.io).

Once published, install with:
```bash
cargo add dotmax
```

## Quick Start

```rust
use dotmax::BrailleGrid;

fn main() {
    // Example coming soon
    println!("dotmax - Terminal braille rendering");
}
```

## Features

- Braille-based rendering for terminal applications
- Image rendering with dithering algorithms
- Drawing primitives (lines, circles, rectangles)
- Color support with customizable schemes
- Animation and frame management
- High-performance rendering optimized for terminal output
